# AgentPrizm JavaScript examples

These examples use Node 18+ native `fetch`; no SDK or dependencies are required.

```bash
export AGENTPRIZM_API_KEY=ap_xxxxxxxxxxxxxxxxxx
export AGENTPRIZM_CONTAINER=demo-acme
npm run quickstart
```

Available examples:

```bash
npm run quickstart
npm run support
npm run sales
npm run coding
```
