# Remote MCP examples

AgentPrizm exposes a remote MCP endpoint:

```text
https://agentprizm.com/api/mcp
```

Use these files as copyable config templates. Replace `ap_xxxxxxxxxxxxxxxxxx` with an API key from your AgentPrizm dashboard.

Available templates:

- `claude-code.json`
- `cursor-mcp.json`
- `claude-desktop.json`

Keep your real API key out of git.
